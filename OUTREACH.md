# OUTREACH PLAN — 研究者・ジャーナリストへのアプローチ

> 方針: 特定個人の名指しリストは作らない（プライバシーとアカウント安全のため）。
> 代わりに「どんな属性の人が反応しやすいか」「どこで見つけるか」「何を送るか」を定義する。
> 鉄則: 宣伝ではなく **フィードバック依頼**。人は宣伝に冷たく、専門性を頼られると答える。

## 1. 反応しやすい相手のプロファイル（優先順）

### A. 計算社会科学・エージェントベースモデル（ABM）系の研究者 — 最優先
- なぜ: このアプリはABMと社会シミュレーションの教育版そのもの。モデルの妥当性を語る言葉を持ち、
  面白ければ授業・ゼミ・学会で使ってくれる（=引用と拡散の質が最も高い）。
- 見つけ方: 計算社会科学会・人工知能学会などの大会プログラムの発表者、
  「計算社会科学」「エージェントベース」「社会シミュレーション」でのX検索、
  大学のシラバス検索（社会シミュレーション/複雑系の講義担当者）。
- 反応しやすいサイン: 自作の可視化やシミュレーションを日頃から投稿している /
  教育・アウトリーチに言及がある / 学生向けの教材を公開している。

### B. 防災・地域政策・インフラ維持系の研究者/実務家
- なぜ: PAGE 2（地域インフラ・後継者ストック・ヘリ3状態）は彼らの現場感と直結。
  縮退運用・スマートシュリンク・中山間地の話題に反応する人は具体的な改善提案をくれる。
- 見つけ方: 地域安全学会・土木計画学系の発表者、自治体シンクタンク・地域おこし系の発信者、
  「インフラ維持」「縮退」「小さな拠点」での検索。

### C. 科学コミュニケーター・教育系の発信者
- なぜ: 「難しい構造を体験で伝える」ことが本業。紹介の文脈を自分で作ってくれる。
- 見つけ方: サイエンスコミュニケーション系イベントの登壇者、教育系YouTube/ブログの運営者、
  高校「公共」「情報I」の実践を投稿している教員。

### D. データジャーナリスト・解説系記者
- なぜ: 選挙や社会分断を「構造」で解説する記事の題材・ビジュアル素材としてアプリが使える。
- 注意: 進行中の政局に紐づけた売り込みはしない（利用は相手の判断に委ねる）。
  「政局コンテンツ」ではなく「構造の教育ツール」として紹介する。
- 見つけ方: データ可視化・ファクトチェック・メディアリテラシー分野の署名記事の書き手、
  データジャーナリズム系の勉強会・アワードの登壇者。

### E. 避けるべき相手
- 政治的立場の強いインフルエンサー（どちらの陣営でも）: アプリが陣営の道具として消費される。
- 炎上系・切り抜き系アカウント: P4で遮断したい外部ノードそのもの。

## 2. アプローチの作法

1. **公開の接点を先に作る**: いきなりDMせず、相手の投稿・論文・記事への実のあるリプや引用を
   1〜2回してから連絡する。「知らない人」からの連絡を「見たことある人」からの連絡に変える。
2. **DM/メールは個別に書く**: テンプレのコピペ感が出た時点で負け。相手の具体的な仕事
   （論文名・記事名・授業名）に必ず1行触れる。
3. **数は絞る**: 週に3〜5人まで。20人に雑に送るより5人に丁寧に送る方が返信率も質も高い。
4. **お願いは1つだけ**: 「10分触って、モデルの粗を1つ指摘してほしい」。拡散のお願いはしない
   （良ければ勝手に共有してくれる）。
5. **もらったフィードバックは必ず反映して報告する**: 「先日のご指摘を反映しました」の一往復が、
   単発の接点を関係に変える。

## 3. メッセージ雛形（必ず個別に書き換える）

### 研究者向け（日本語・DM/メール共通の骨格）
```
はじめまして。[相手の論文/発表/記事名] を拝見し、[具体的に何が参考になったか1行]。

社会や地域インフラが「どう壊れるか」を体験できる教育用シミュレーターを
個人で開発しています（非営利・ブラウザで開くだけ・約60秒で試せます）:
[URL]

[相手の専門] のご視点で、モデルの粗をひとつだけご指摘いただけないでしょうか。
特に [そのひとに関係する具体的な部分: 例. PAGE 2の後継者ストックの力学] の
妥当性についてご意見を伺えたら幸いです。

10分ほどで触れる規模です。お忙しいところ恐れ入りますが、
もしご興味が向いたときにでも。
```

### 教員向け
```
[授業実践の投稿など]を拝見しご連絡しました。
「社会がどう壊れるか」をスライダー操作で体験できる無料の教材アプリを作っています。
準備物なし・ブラウザのみ・1テーマ10分で使えます: [URL]
授業で使うとしたら何が足りないか、教える側の視点でご意見をいただけないでしょうか。
```

### ジャーナリスト向け
```
[相手の記事名]を拝読しました。[具体的な感想1行]。
分断や地域インフラの崩壊を「個人ではなく構造」で体験的に理解するための
教育用シミュレーターを非営利で公開しています: [URL]
記事の題材や読者向けの補助教材になり得るか、ご感想をいただけたら嬉しいです。
```

### 英語版（研究者向け・Show HN 前の個別連絡にも流用可）
```
Hi — I came across your work on [specific paper/project] and found [1 specific line].
I'm building a free, non-commercial educational simulator that lets people
experience *how* societies and regional infrastructure fail (browser-only, ~60s to try): [URL]
Would you be willing to point out one flaw in the model from your perspective?
I'd especially value your view on [specific mechanism].
```
