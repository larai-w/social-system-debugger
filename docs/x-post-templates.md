# X 投稿文テンプレ集 — 毎週「埋めるだけ」の運用キット

> MARKETING.md のコンテンツカレンダー（月・木・日）に対応。`{{ }}` を実際の値に置き換えて投稿する。
> 数字（生存率など）は Plausible / アプリ内 track イベントの週次集計から拾う。

## 全投稿に共通の固定ルール（コピペ前に毎回確認）

1. ハッシュタグは **`#社会デバッガー`** を1つだけ（アプリ内の共有と同じ固定タグ。乱立させない）。
2. URL は常に1本: `https://larai-w.github.io/social-system-debugger/`
3. 実在の地名・人名・進行中の政局に触れない。抽象型（H型/T型・「ある街」）で語る。
4. 末尾（またはスレッド末尾）に免責1行: **※特定の誰かではなく、どの社会にも起こる構造の話です**
5. 動画・GIF を添える場合、リール由来なら **「※演出用に簡略化したデモ映像です」** を必ず添える。
6. 宣伝ツイートを拡散しない。**面白い結果そのもの**をコンテンツとして流す。
