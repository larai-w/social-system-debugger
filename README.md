# [ SOCIAL SYSTEM DEBUGGER v2.0 ]

**社会崩壊のメカニズムを、コードの言葉で理解する。**

ポピュリズム・エコーチェンバー・倫理の欠如がなぜ社会を壊すのかを、機械学習とアルゴリズムの比喩でリアルタイムにシミュレートするインタラクティブWebツール。

🔗 **[ライブデモ](https://larai-w.github.io/social-system-debugger/)**

---

## 何ができるか

3つのパラメーターを操作して、社会システムのリアルタイム変化を観察します。

| パラメーター | 範囲 | 対応する概念 |
|---|---|---|
| 情報フィルタリング率 | 0%（オープン）→ 100%（エコーチェンバー） | 機械学習の**過学習（Overfitting）** |
| リーダーの倫理観 | 100（法の支配）→ 0（ポピュリズム） | 報酬関数のドーパミン最適化 |
| 意思決定アルゴリズム | 動的計画法（DP）↔ Greedy | 長期最適化 vs 局所最適解 |

操作に応じて4つのビジュアライゼーションがリアルタイムで更新されます。

---

## 4つの可視化

**① DATA GENERALIZATION（散布図）**
フィルタリング率が上がると多様な本物データ（青）が消え、エコーチェンバー的な過学習データ（赤）のクラスターへとドリフトしていく様子を観察できます。

**② SOCIAL HEALTH TIMELINE（時系列グラフ）**
Greedy × 低倫理の組み合わせで短期ドーパミン（熱狂度）が爆発的に上昇したあと、ソーシャルキャピタルとインフラが段階的に崩壊し `SYSTEM CRASH` アラートが発動します。

**③ SYSTEM STABILITY MATRIX（レーダーチャート）**
信頼・回復力・正当性・情報健全性・長期持続性・精神的健全性の6軸。赤い破線（Failure Threshold）を下回ると崩壊域に入ります。

**④ AGENT NETWORK SIMULATION（エージェント可視化）**
48エージェントが情報を伝播するリアルタイムアニメーション。フィルタリング率が上がるにつれてクロスビリーフのエッジが切断され、信念クラスターが形成されていきます。

## 試してみるシナリオ

**「ワイマール崩壊を再現する」**
→ フィルタリング率 88% / 倫理観 12 / Greedy を選択。ドーパミンが跳ね上がったあと、すべての指標が崩壊するまで待つ。

**「なぜ北欧はうまくいっているのか」**
→ 北欧型民主主義プリセットを選択（フィルタリング率 18% / 倫理観 90 / DP）。レーダーチャートがほぼ完全な形を保ち続けることを確認する。

**「SNS社会の構造的問題」**
→ SNS社会 2024 プリセット。倫理観は中程度なのにフィルタリング率が高いと、ゆっくりと確実に劣化することが見える。

## 何が学べるか

- **過学習の社会的類比** — エコーチェンバーがなぜ機械学習の過学習と同じ構造を持つのか
- **Greedyアルゴリズムの罠** — 短期最適化が長期破綻を引き起こすメカニズム
- **ソーシャルキャピタル理論** — 信頼と制度がシステム耐久性を決める理由
- **エントロピーと社会秩序** — 情報理論の無秩序度が社会崩壊の先行指標になる理由
- **エージェントベースモデル** — Markov Chain + Monte Carlo + ネットワーク伝播による社会シミュレーション
