社会デバッガー

あなたの街は、生き残れるか。 — 社会や地域インフラが「どう壊れるか」をスライダー操作で体験する教材シミュレーター

▶ https://larai-w.github.io/social-system-debugger/
  • 準備物なし（ブラウザのみ）
  • 無料・広告なし・登録なし
  • 1テーマ約10分
  • スマホ・タブレット・PC対応
  • 日本語 / English
  • 個人を特定しない計測のみ

1. 5分で始める

  1. 上のURLを開く（最初の説明はスキップ可）。
  2. 上部タブから今日のテーマ（下表の4ページ）を1つ選ぶ。
  3. プリセットを1タップ → 社会が動き出す → スライダーを動かして「介入」してみる。

2. 4つのテーマと授業での問いの例

ページ体験できる構造問いの例
PAGE 1
情報空間		フィルターバブルと過学習。情報の偏りが多様性と信頼を静かに削る過程 「見たい情報だけを見ると、なぜ社会全体が壊れやすくなるのか？」
PAGE 2
地域インフラ		人口減少・効率化・後継者不足。効率を上げたのに災害に弱くなる逆説 「効率化はいつ正解で、いつ命取りになるか？（冗長性の価値）」
PAGE 3
認知リカバリー		怒りが止まらなくなる認知の仕組みと、そこからの回復 「怒りが止まらないのは意志の弱さか、それとも構造か？」
PAGE 4
ステークホルダー		当事者の声が観客の歓声にかき消されていく通信路の力学 「当事者の声が届かなくなるのは、誰が悪いときか？ 誰も悪くないときか？」

探究の問い: 「崩壊した街を1つだけ過去に戻して変えられるなら、あなたは何を守るか？」（PAGE 1 の巻き戻し機能で実際に試せる）

3. 授業の型

10分（導入）教員がプリセット1つを投影 → 崩壊を見せる → ペアで問いを1つ議論。
50分（1コマ）①各自で1テーマ体験（10分）→ ②「生き残る設定」の仮説づくり（10分）→ ③パラメータ実験で検証（15分）→ ④班で共有: どの介入が効いたか・現実では何にあたるか（15分）。
探究・課題研究メニューの「📊 データを書き出す (JSON/CSV)」で全パラメータと結果を保存し、条件を変えた比較実験・レポート作成に使える。共有URLは実験の再現に使える。

4. 扱い方の注意（大切）

このアプリは架空のシミュレーションです。実在の地名・人名・進行中の政治的話題は登場しません。 授業でも「特定の誰か・どこかの話」ではなく「どの社会にも起こりうる構造の話」として扱うことを推奨します。 結果を根拠に現実の特定の地域・人物・政党を評価する使い方は、このモデルの精度を超えています。
無料・非営利・オープンソース（GitHub: larai-w/social-system-debugger）。 授業で使ってみて足りないものがあれば、アプリ内「💬 フィードバック」からお知らせください。教える側の視点でのご意見が一番の改善材料になります。
