社会デバッガー — プライバシーポリシー

短くまとめると: アカウント不要。個人情報を集めません。あなたの記録は、あなたの端末の中だけに残ります。

1. 収集しない情報

このアプリを使うのにアカウント登録は不要です。氏名・メールアドレス・電話番号などの個人情報を、こちらから収集することはありません。

  • 例外として、アプリ内の「💬 フィードバック」からご連絡をいただく際に、あなたが任意でメールアドレスを入力した場合のみ、そのアドレスを返信のためだけに使います。それ以外の目的には使いません。

2. 端末内に保存する情報

設定・街の名前・発見ログ・週次クリアの記録などは、ブラウザの localStorage（ネイティブ版はアプリの Preferences にも複製）に、あなたの端末内にのみ保存されます。

  • これらのデータがこちらのサーバーに送信されることはありません。
  • ブラウザのサイトデータを削除する、またはアプリを削除すると、これらの記録は消えます。

3. アナリティクス（利用状況の集計）

現在、アナリティクスは導入していません。訪問数を含め、何も計測していません。

  • 将来、Cookie を使わない匿名の集計計測（Plausible などの、個人を特定しない仕組み）を有効にする可能性があります。その場合は、このページを更新してお知らせします。

4. 第三者サービス

  • グラフ描画ライブラリ（Chart.js）を CDN（外部の配信ネットワーク）から読み込みます。そのため、CDN の事業者側に、通常の Web アクセスと同様のログ（IPアドレス等）が残り得ます。
  • X・LINE への共有は、あなたが共有ボタンを明示的に押したときにのみ行われ、それぞれのサービスの規約のもとで処理されます。押さなければ、これらのサービスに情報は渡りません。

5. 通知（ネイティブ版のみ）

ネイティブ版（iOS / Android アプリ）の通知は、週1回のローカル通知だけです。端末の中で完結し、端末の外へ情報が出ることはありません。

  • 通知の許可は、いつでも OS の設定から取り消せます。

6. お問い合わせ

プライバシーに関するご質問は、アプリ内の「💬 フィードバック」、または GitHub リポジトリ （larai-w/social-system-debugger の Issues）からお寄せください。

このアプリは無料・非営利のオープンソースです。上に書いたこと以上のデータは扱っていません。 「特定の誰かの情報を集める」のではなく、「どの社会にも起こる構造」を体験してもらうことが目的のアプリです。
