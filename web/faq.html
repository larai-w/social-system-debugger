社会デバッガー — よくある質問（FAQ）

短くまとめると: 無料・登録なし。ブラウザでそのまま使えます。ホーム画面に入れれば全画面＆オフラインでも動きます。

1. 「ブラウザ版」「PWA版」「ストア版」は何が違う?

中身は同じシミュレーターです。「どうやって起動するか」だけが違います。今日すぐ選べるのは ブラウザでそのまま と ホーム画面に入れる（PWA） の2形態。ストアアプリ版は準備中です。

ブラウザでそのまま ホーム画面に入れる（PWA） ストアアプリ
インストール 不要 ブラウザから「ホーム画面に追加」/「インストール」 —
起動方法 URL を開く ホーム画面のアイコンから、アプリのように —
オフライン ◯（一度開けば動く） ◯（一度開けば動く） —
全画面 ブラウザのタブ内 ◯ 全画面で起動 —
費用 無料 無料 —
提供状況 今すぐ使える 今すぐ使える 準備中

ホーム画面への入れ方（PWA）:

  • iPhone / iPad（Safari）: 共有ボタン（□に↑）→「ホーム画面に追加」
  • Android（Chrome）: メニュー（⋮）→「アプリをインストール」
  • PC（Chrome / Edge）: アドレスバー右端のインストールアイコン、またはメニュー→「インストール」

2. データはどこに保存されますか?

設定・街の名前・記録などは、ブラウザの localStorage に、あなたの端末内にのみ保存されます。こちらのサーバーには送信されません。

  • ≡メニューの「エクスポート」から、パラメータや指標を JSON / CSV で持ち出せます。
  • ブラウザのサイトデータを削除すると、これらの記録は消えます。
  • 詳しくは プライバシーポリシー をご覧ください。

3. オフラインで動きますか?

はい。一度オンラインで開けば、Service Worker がページとスクリプトをキャッシュするので、その後はオフラインでも動きます。ホーム画面に入れておく（PWA）と、より安定して起動できます。

  • グラフ描画ライブラリ（Chart.js）は外部 CDN から読み込みます。読み込めない環境でも、スライダー・タブ・共有などの主要機能は動きます（グラフだけが省略されます）。

4. 無料ですか? 登録は必要ですか?

  • 無料です。料金は一切かかりません。
  • 登録・アカウントは不要です。メールアドレスなどの入力もいりません。
  • 広告はありません。
  • オープンソースです（larai-w/social-system-debugger）。

5. 授業や研究で使えますか?

はい。非営利・教育目的での利用を歓迎します。次の資料があります。

  • 授業で使える1枚ガイド（A4印刷 → PDF）
  • 方法論の説明: METHODOLOGY.md
  • エクスポート項目の対応表: DATA-DICTIONARY.md
